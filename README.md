# TheNotesApp

## General info
TheNotesApp is a note taking application with cool note organization features which helps user to take effective notes.
	

## Technologies
Project is created with:

* Kotlin
* Jetpack Compose
* MVVM
* Room
* Dagger-HILT
* Coroutines

## Features

* Sort By Title,Color,Date
* Different colors to choose from
